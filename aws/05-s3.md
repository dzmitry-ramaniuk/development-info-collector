## Базовые возможности S3

**S3** — объектное хранилище с практически неограниченной масштабируемостью.

Типовые use cases:
- пользовательские файлы;
- бэкапы и архивы;
- статика для web/CDN;
- data lake и логи.

## Безопасность доступа

- IAM policies + bucket policies.
- Block Public Access включён по умолчанию.
- Pre-signed URL для временного доступа.
- SSE-S3 или SSE-KMS для шифрования.

## Надёжность и защита данных

- Включайте versioning.
- MFA delete для критичных бакетов.
- Cross-Region Replication для DR-сценариев.
- Object Lock для WORM-требований.

## Оптимизация стоимости

- Lifecycle policies для перехода в более дешёвые классы хранения.
- Удаление старых версий и incomplete multipart uploads.
- Анализ паттернов через S3 Storage Lens.

## Практический baseline для production

1. Versioning + encryption.
2. Запрет публичного доступа.
3. Lifecycle для архивирования.
4. Логи доступа и мониторинг аномалий.
5. Отдельные bucket’ы по назначению и окружениям.
