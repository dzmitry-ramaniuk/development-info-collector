# Тестирование: руководства и практики

Этот каталог содержит материалы по различным аспектам тестирования программного обеспечения. Каждый файл — самостоятельное руководство, которое можно использовать для изучения инструментов тестирования, подготовки к собеседованиям или в повседневной работе.

## Структура материалов
- [JUnit — фреймворк модульного тестирования для Java](01-junit.md)

Материалы включают теоретические основы, практические примеры кода и вопросы для самопроверки. Используйте их как краткий справочник для освоения инструментов тестирования и обеспечения качества кода.
