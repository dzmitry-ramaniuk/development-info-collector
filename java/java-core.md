# Java Core: структура материала

Java Core — фундамент платформы, на котором строится большинство экосистемных технологий (Spring, Android, Big Data). Чтобы системно освоить язык, следуйте по главам в указанном порядке:

1. [Виртуальная машина и платформа](java-core/jvm-runtime.md) — как устроена JVM, какие роли выполняют JDK и JRE, что происходит при запуске программы.
2. [Управление памятью и сборка мусора](java-core/memory-management.md) — устройство кучи, алгоритмы сборки мусора, анализ утечек.
3. [Основы языка и синтаксис](java-core/language-basics.md) — типы данных, управление потоком, модули и базовые конструкции.
4. [Объектно-ориентированное программирование](java-core/oop-design.md) — классы, интерфейсы, принципы проектирования и шаблоны.
5. [Коллекции и обобщения](java-core/collections-generics.md) — структура `java.util`, generics и идиомы эффективной работы.
6. [Работа с данными и ввод/вывод](java-core/data-io.md) — классический I/O, NIO.2, сериализация, API даты и времени.
7. [Функциональные возможности и современные фичи](java-core/functional-modern-java.md) — lambda-выражения, Stream API, Optional, эволюция языка.

> Каждая глава содержит развёрнутый конспект и блок «Вопросы на собеседовании», чтобы закрепить материал и подготовиться к интервью.
