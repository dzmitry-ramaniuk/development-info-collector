# Java Core: структура материала

Java Core — фундамент платформы, на котором строятся все стековые решения: от Android до серверных приложений и big data.
Конспект разбит на главы, которые можно читать последовательно или точечно при повторении определённой темы. Каждая глава
заканчивается блоком «Вопросы на собеседовании», а внутри вы найдёте схемы, практические советы и ссылки на дополнительные
источники.

## Дорожная карта глав
1. [Виртуальная машина и платформа](java-core/01-jvm-runtime.md) — устройство JVM, жизненный цикл программы, инструменты
   диагностики.
2. [Управление памятью и сборка мусора](java-core/02-memory-management.md) — память процесса, типы сборщиков, настройка и
   отладка утечек.
3. [Основы языка и синтаксис](java-core/03-language-basics.md) — типы данных, управление потоком, модульная система, работа с
   исключениями.
4. [Объектно-ориентированное программирование](java-core/04-oop-design.md) — классы, интерфейсы, SOLID, паттерны, проектирование
   API.
5. [Коллекции](java-core/05-collections.md) — структура `java.util`, List, Set, Queue, Map, детали реализаций, 
   итераторы, сортировка, контракты equals/hashCode.
6. [Работа с данными и ввод/вывод](java-core/06-data-io.md) — эволюция I/O API, NIO.2, работа с файлами, сериализация и парсинг.
7. [Функциональные возможности и современные фичи](java-core/07-functional-modern-java.md) — lambda, функциональные интерфейсы, Optional, новые
   возможности языка вплоть до актуальной версии.
8. [Алгоритмическая сложность и анализ производительности](java-core/08-algorithms-complexity.md) — O-нотация, временная и
   пространственная сложность, анализ алгоритмов с акцентом на логарифмическую сложность.
9. [Stream API](java-core/09-stream-api.md) — подробное описание Stream API: создание стримов, промежуточные и терминальные операции, коллекторы, параллельные стримы.
10. [Типы ссылок](java-core/10-reference-types.md) — Strong, Weak, Soft, Phantom ссылки, ReferenceQueue и практическое применение.
11. [Обобщения (Generics)](java-core/11-generics.md) — типобезопасность, type erasure, wildcards, PECS принцип, продвинутые техники.

> **Как читать главы.** Перед погружением в узкую тему освежите знания об устройстве JVM и памяти: понимание рантайма помогает
> объяснить поведение языка. После прочтения практикуйтесь — главы содержат упражнения и идеи мини-проектов.
