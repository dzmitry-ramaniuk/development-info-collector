# Spring Core

## Ключевые концепции
- Инверсия управления (IoC) и внедрение зависимостей (DI).
- Конфигурация через Java, XML и аннотации.
- Жизненный цикл бинов и область их видимости.

## Практические заметки
- Используйте `ApplicationContext` для управления зависимостями и ресурсами.
- Разделяйте конфигурацию на модули, чтобы упростить поддержку и тестирование.
