# Общие инструкции для агентов

- В тематических каталогах файлы с материалами должны иметь числовой префикс формата `NN-` (две цифры и дефис), отражающий порядок изучения.
- Если в каталоге появляется новый файл без префикса, его нужно пронумеровать, подобрав следующий свободный номер и обновив ссылки.
- Обзорные файлы `README.md` и служебные `.gitkeep` префиксами не нумеруются.
- После переименования обязательно обновляйте все внутренние и перекрёстные ссылки.
